# Expense Tracker - Modern Android Build

A clean, minimalist expense tracking application built to demonstrate **Modern Android Development (MAD)** practices.

## 🛠 Tech Stack
- **UI:** Jetpack Compose (Material 3)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Database:** Room Persistence Library
- **Async:** Kotlin Coroutines & Flow
- **Navigation:** Jetpack Navigation Compose

## 🚀 Features
- **Persistent Storage:** Expenses are saved locally using Room.
- **Reactive UI:** The dashboard updates automatically when data changes using StateFlow.
- **Categorization:** Custom icons and colors for different spending types.
- **Calculations:** Real-time summary of total expenses.

## 📲 How to Install & Try the App

Since this app is currently in development and not yet on the Google Play Store, you can install the latest version directly from this repository:

1. **Download the APK:** Go to the [Releases](link-to-your-repo-releases) section of this GitHub and download `app-debug.apk`.
2. **Transfer to Phone:** Move the file to your Android device (via USB, Google Drive, or Telegram).
3. **Enable Unknown Sources:** - Open your phone **Settings**.
   - Go to **Security** or **Apps**.
   - Enable **"Install unknown apps"** for your browser or file manager.
4. **Install:** Tap the `.apk` file and select **Install**.
5. **Launch:** Open "Expense Tracker" and start managing your budget!

> **Note:** As this is a debug build, your phone might show a "Play Protect" warning. You can safely click "Install Anyway" as this is your own trusted code.
